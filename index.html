TunnelMap — help

modes

eedit (default) — click selects, drag moves, marquee on empty drag
ddraw — click empty extends the active line
llift — two waypoint clicks build a cross-plane connector

core gestures (any mode)

click waypointselect it · make line active
click tubeselect that line · make active
click empty (different plane)switch active plane to that one
drag waypointmove it (octolinear snap)
shift+drag waypointmove between planes
drag emptymarquee select
dbl-click waypointopen station popover
dbl-click tubeinsert bend at cursor
dbl-click emptystart new line here · enter draw
right-click targetcontext menu for that thing
right-click emptydeselect everything

planes

1…6jump to plane index
[/]previous / next plane

transforms

r / shift+rrotate selection 90° cw / ccw
h / vmirror horizontally / vertically
\\straighten active line
tabflip line direction
delremove selected waypoints

cmd+zundo
cmd+shift+zredo
